How should the user set the timeline?

The store saves timelineMonths: number. Which input should the Edit panel expose?

A

Months number input only

Simple. Type "36" for 3 years. No conversion needed. Matches what the store already holds. Completion date shown as a computed label below the input.

B

Date picker only

User picks "June 2029". Months derived from today → that date. Store still saves timelineMonths (converted). More intuitive for goal-oriented thinking.

C

Both, linked together

Date picker + months input side by side. Editing one updates the other live. Most flexible but more complex to build and harder to fit in the 300px panel.
