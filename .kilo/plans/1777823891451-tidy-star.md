## Edge Cases
- **No debts**: Show empty state "You're debt-free! Track your debt here when you have any." with a link to onboarding
- **Single debt**: Strategy comparison is identical â€” show a note "With only one debt, both strategies behave the same."
- **Store data stale**: If `debts.items` exists but `plan` is null, fall back to computing goal allocations from store goals
- **Sankey with 0 values**: Filter out 0-value links to avoid rendering empty flows
- **Sankey color resolution**: Resolve CSS variables to hex at render time (recharts SVG can't parse `var()`)
- **ErrorBoundary key**: Same as existing routes â€” `key={location.pathname}` for auto-reset on navigation
