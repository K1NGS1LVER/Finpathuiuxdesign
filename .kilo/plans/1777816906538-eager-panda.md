## Component Breakdown

### 1. `useJourneyCanvas.ts` — Canvas Interaction Hook
Extract all pointer/pan/zoom/drag logic and state:
- `positions` state, `dragging` state, `dragOffset`, `panOffset`, `isPanning`, `panStart`, `zoom` state
- `getNodePos()` — default positioning (income node left, goals in grid)
- `getPointerPosition()` — unified mouse/touch
- `handlePointerDown(nodeId)` — begin drag on a node
- `handlePointerMove(event)` — update position during drag or pan
- `handleCanvasPointerDown(event)` — begin pan on empty canvas
- `handleWheel(event)` — zoom around pointer
- `handlePointerUp()` — stop drag/pan
- Return: all state + handlers + `getNodePos`, `canvasRef`

### 2. `useJourneyGoals.ts` — Goal Management Hook
Extract all budget/validation/goal-mutation logic:
- Store selectors (goals, income, expenses, debts, monthlySurplusReserve)
- `sortedGoals`, `activeGoals` (useMemo)
- `monthlySurplus`, `existingMonthlyNeed`, `budgetRemaining` calculations
- `handleComplete(goalId)` — calls `completeGoal`, fires confetti, closes detail
- `handleDelete(goalId)` — calls `removeGoal`, closes detail
- `handlePriorityChange(goalId, newPriority)` — reorder logic
- `handleAddPreset(preset)` — budget validation (hard block + soft block), addGoal
- `handleAddCustom()` — same validation + custom fields
- Returns: all data + handlers + `selectedGoalId`, `setSelectedGoalId`
