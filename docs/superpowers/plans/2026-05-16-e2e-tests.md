# E2E Test Suite Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add 22 Playwright E2E tests covering auth routing, all 7 app screens (golden paths + key interactions), sidebar navigation, and Penny chat — all running offline via localStorage seeding and API route mocking.

**Architecture:** Playwright installed in `frontend/`. Each test injects a pre-built `finpath-store` into `localStorage` via `page.addInitScript` before navigation — skipping auth and onboarding. A shared `webServer` config starts Vite with `VITE_AUTH_MOCK=true`. All `/api/*` calls are intercepted by `page.route()`. No backend process needed.

**Tech Stack:** `@playwright/test`, Playwright Chromium, React 18 SPA (Vite, localhost:5173), Zustand localStorage store (key: `finpath-store`, version 5)

import type { Page } from '@playwright/test';

/** Mock POST /api/penny — returns a fixed non-streaming reply. */
export async function mockPenny(page: Page): Promise<void> {
  await page.route('**/api/penny', (route) => {
    route.fulfill({
      status: 200,
      contentType: 'application/json',
      body: JSON.stringify({
        reply:
          'Your savings rate is 25%. Consider building your emergency fund to 6 months of expenses.',
      }),
    });
  });
}

/** Mock POST /api/penny/stream — returns a minimal SSE sequence. */
export async function mockPennyStream(page: Page): Promise<void> {
  await page.route('**/api/penny/stream', (route) => {
    const body = [
      'event: token\ndata: {"token":"Your savings rate looks healthy."}\n\n',
      'event: done\ndata: {}\n\n',
    ].join('');
    route.fulfill({
      status: 200,
      contentType: 'text/event-stream',
      body,
    });
  });
}

/** Mock all /api/simulate/* endpoints — return empty-but-valid shapes. */
export async function mockSimulate(page: Page): Promise<void> {
  await page.route('**/api/simulate/**', (route) => {
    route.fulfill({
      status: 200,
      contentType: 'application/json',
      body: JSON.stringify({}),
    });
  });
}

## Troubleshooting

**Sidebar collapsed:** Add `viewport: { width: 1280, height: 900 }` to `playwright.config.ts` `use` block.

**Store not applying:** Check `localStorage.getItem('finpath-store')` in browser console during debug. Ensure `version: 5` matches the current persist version in `store.ts`.

**Wrong mock route:** FinPath uses both `/api/penny` (non-streaming) and `/api/penny/stream` (SSE). Check which one `PennyPanel.tsx` calls in the code path being tested and mock the right one.

**Auth redirect loop:** With `VITE_AUTH_MOCK=true`, `auth-store.ts` immediately sets a mock user. If the `finpath-store` seed has `onboarded: false`, the router sends to `/onboarding`. Always use `seedAndGo` (which injects `onboarded: true`) for app-screen tests.
